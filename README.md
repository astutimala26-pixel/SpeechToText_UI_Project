This is a web-based Speech-to-Text Dashboard that converts live voice or audio files into text in real-time using the Web Speech API.
The application provides a simple interface for recording speech and viewing transcribed text instantly.
It also includes a Statistics Tracker that records total words spoken, number of sessions, and average session duration.
The statistics are stored using localStorage, allowing data to persist even after refreshing the browser.

Technologies Used:
HTML
CSS
JavaScript
Web Speech API
LocalStorage

Features:
Real-time speech-to-text transcription
Audio file transcription
Live text display
Session statistics tracking
Persistent data storage
